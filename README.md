# TechFix — IT Support Ticket Dashboard

A modern, responsive IT support ticket dashboard built with pure HTML, CSS, and JavaScript. Users can submit IT-related issues, manage ticket statuses, filter tickets, and view real-time dashboard statistics.

## Technologies Used

- **HTML5** — Semantic markup
- **CSS3** — Modern styling with CSS variables, glassmorphism, gradients, and responsive design
- **JavaScript (Vanilla)** — Client-side logic, localStorage persistence, DOM manipulation

## Features

- Ticket submission form with name, email, issue type, priority, and description
- Auto-generated ticket IDs (e.g., TCK-1001)
- Status management — Pending, In Progress, Resolved
- Priority badges — Low, Medium, High
- Filter tickets by status or high priority
- Real-time dashboard statistics (total, pending, in progress, resolved)
- Delete tickets with confirmation
- Dark / Light mode with persistent preference
- LocalStorage persistence — tickets survive page refresh
- Responsive layout — mobile, tablet, and desktop
- Empty state messaging
- Professional footer with developer credit

## How to Run

1. Clone or download the project folder.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
3. No server, build step, or dependencies required.
